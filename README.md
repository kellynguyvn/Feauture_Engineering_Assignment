# Feauture Engineering Assignment
## Learning Objective
The purpose of the assignment is to derive feature engineering spectrograms of wave files. Included are the audio files of Kepler: Star KIC12268220C Light Curve Waves to Sound, NASA - Kepler: Star KIC7671081B Light Curve Waves, NASA - Whistler Waves, Johns Hopkins APL - Parker Solar Probe - Whistler Mode Waves 2, and NASA - Audio from NASAâ€™s Juno Mission: Europa Flyby. The Librosa library was used to analyze the audio data.


## What each file contains:
1. Displays a spectrogram/chromagram/cqt/etc. 
2. Computes a mel-scaled spectrogram 
3. Short-time Fourier transform (STFT). 
4. beat tracker 
5. constant-Q transform of an audio signal.
